## 実行コマンド（要点）
### Git 初期設定
```bash
git --version
git config --global user.name "tomo44584458"
git config --global user.email "tomo44584458@ymail.ne.jp"
git config --global init.defaultBranch main
git config --global pull.rebase false
git config --global --list
```
### SSH鍵作成 → GitHub 接続確認
```bash
ssh-keygen -t ed25519 -C "（メール）"
cat ~/.ssh/id_ed25519.pub
ssh -T git@github.com
```

### リポジトリ初期化〜push（作業フォルダで）
```bash
git init
git remote -v
git push -u origin main
```
